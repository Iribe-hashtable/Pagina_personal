Proyectos

Buscador de frases

Facilita a los usuarios la búsqueda de citas célebres utilizando palabras clave. Estas citas pueden ser utilizadas en redes sociales, presentaciones o para identificar su fuente original. Python se emplea como lenguaje de programación para acelerar este proceso de búsqueda de información.

E-commerce

Base de datos escolar
